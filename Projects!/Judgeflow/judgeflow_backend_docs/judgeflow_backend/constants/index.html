§Constants Module

This module defines application-wide constants used throughout the Judgeflow backend. It includes environment variable names for configuration and other compile-time constants.

§Usage

Import the constants as needed in your modules:

use crate::constants::{
    SUPABASE_URL, 
    SUPABASE_API_KEY, 
    SUPABASE_SERVICE_ROLE_KEY,
    ALLOWED_ORIGINS
};

§Environment Variables

These constants should be set in your environment or .env file with appropriate values:

  • SUPABASE_URL: The base URL of your Supabase project
  • SUPABASE_API_KEY: The public API key for Supabase
  • SUPABASE_SERVICE_ROLE_KEY: The service role key for admin operations (keep this secure!)

§Runtime Constants

  • ALLOWED_ORIGINS: A set of allowed CORS origins for the application
