pub fn read_csv_from_buffer(

Expand description

Reads and parses CSV data from a byte buffer into a vector of hash maps.

This function takes a byte buffer containing CSV data and parses it into a more accessible format where each row is represented as a HashMap with column headers as keys and row values as string values.

ยงArguments

  • buffer - A byte slice containing the CSV data to parse

ยงReturns

  • Result<Vec<HashMap<String, String>>, CsvProcessingError> - A result containing either:
    • Ok(Vec<HashMap<String, String>>) - A vector of hash maps, where each map represents a row
    • Err(CsvProcessingError) - An error if parsing fails

ยงErrors

Returns a CsvProcessingError if:

  • The input is not valid UTF-8
  • The CSV is malformed
  • Headers cannot be read
  • Rows have inconsistent numbers of fields

ยงExample

use std::collections::HashMap;
use judgeflow_backend::services::import_service::read_csv_from_buffer;

let csv_data = b"name,age\nAlice,30\nBob,25";
let result = read_csv_from_buffer(csv_data).unwrap();
 
let mut expected = vec![
    {
        let mut m = HashMap::new();
        m.insert("name".to_string(), "Alice".to_string());
        m.insert("age".to_string(), "30".to_string());
        m
    },
    {
        let mut m = HashMap::new();
        m.insert("name".to_string(), "Bob".to_string());
        m.insert("age".to_string(), "25".to_string());
        m
    }
];

assert_eq!(result, expected);
