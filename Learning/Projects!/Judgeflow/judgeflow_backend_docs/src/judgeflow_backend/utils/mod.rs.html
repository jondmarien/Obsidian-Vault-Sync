1//! # Utilities Module
2//!
3//! This module contains various utility functions and types used throughout the Judgeflow backend.
4//! It's organized into submodules, each focusing on a specific area of functionality.
5//!
6//! ## Available Submodules
7//!
8//! - `csv_processing`: Tools for processing CSV files, including reading, writing, and transforming CSV data.
9//! - `eligibility`: Functionality for checking participant eligibility based on various criteria.
10//!   - `eligibility_checks`: Core logic for individual eligibility checks.
11//!   - `eligibility_csv`: Batch processing of eligibility checks from CSV files.
12//!
13//! ## Usage
14//!
15//! Import the specific utilities you need in your module:
16//!
17//! ```rust
18//! // For CSV processing
19//! use judgeflow_backend::utils::csv_processing::{read_csv, write_csv};
20//!
21//! // For eligibility checking
22//! use judgeflow_backend::utils::eligibility::{
23//!     EligibilityCheckInput,
24//!     eligibility_checks::perform_all_checks,
25//!     eligibility_csv::check_eligibility_from_csv_and_save
26//! };
27//! ```
28//!
29//! ## Features
30//! - **CSV Processing**: Read and write CSV files with automatic type conversion
31//! - **Eligibility Checking**: Validate participant eligibility based on configurable rules
32//! - **Batch Processing**: Process multiple participants efficiently from CSV files
33//!
34//! ## Error Handling
35//! Most functions in this module return `Result` types with appropriate error variants.
36//! Always handle potential errors when using these utilities.
