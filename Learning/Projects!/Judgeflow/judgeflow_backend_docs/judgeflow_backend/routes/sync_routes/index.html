§User Synchronization Routes

This module provides endpoints for synchronizing user data between Clerk (authentication provider) and the application’s database. This ensures user data consistency across services.

§Endpoints

  • POST /api/sync-users: Synchronizes all users from Clerk to the application database

§Authentication

These endpoints typically require admin-level authentication.

§Rate Limiting

These endpoints may be rate-limited to prevent abuse.
