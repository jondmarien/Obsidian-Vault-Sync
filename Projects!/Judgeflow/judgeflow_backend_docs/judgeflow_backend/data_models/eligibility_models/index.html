§Eligibility Check Models

This module defines the data structures used for participant eligibility verification in hackathons. It includes input validation, result tracking, and criteria evaluation.

§Key Components

  • EligibilityCheckInput: Participant data for eligibility verification
  • EligibilityCriterionResult: Individual criterion evaluation result
  • EligibilityCheckResult: Aggregated eligibility assessment
  • CriterionStatus: Status enum for evaluation results

§Usage

use judgeflow_backend::data_models::eligibility_models::{
    EligibilityCheckInput,
    EligibilityCheckResult,
    CriterionStatus
};
