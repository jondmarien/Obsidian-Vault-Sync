§CSV Processing Module

This module provides comprehensive functionality for processing CSV files in the Judgeflow application. It includes utilities for reading, validating, normalizing, and transforming CSV data into structured Rust types.

§Features

  • CSV Reading: Flexible CSV parsing with support for different formats and encodings
  • Data Normalization: Standardize data like country names, dates, and names
  • Project Processing: Specialized handling for project-related data
  • Data Validation: Ensure data integrity and consistency
  • Pretty Printing: Format data for console output

§Quick Start

use std::path::Path;
use judgeflow_backend::utils::csv_processing::{
    read_csv_records,
    project_processing::process_projects_csv,
    registrant_normalization::normalize_and_print_registrants,
};

// Read any CSV into a vector of records
let registrants: Vec<HashMap<String, String>> = read_csv_records("data/registrants.csv")?;
 
// Process project data
let projects = process_projects_csv(Path::new("data/projects.csv"))?;
 
// Normalize and print registrant data
normalize_and_print_registrants("data/registrants.csv")?;

§Modules

  • country_normalization: Standardize country names and codes
  • fix_csv: Repair malformed CSV files
  • project_processing: Handle project and team member data
  • registrant_normalization: Process and normalize registrant information
