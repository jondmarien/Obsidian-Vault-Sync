§Devpost Integration Models

This module defines data structures for interacting with Devpost’s API and processing Devpost-related data. It includes models for projects, user profiles, and scraping operations.

§Key Components

  • DevpostProject: Represents a project imported from Devpost
  • DevpostScrapeSseRequest: Parameters for initiating a Devpost profile scrape
  • DevpostScrapeResult: The result of a Devpost profile scraping operation
  • DevpostUserLookupResult: Information about a user’s Devpost data status

§Usage

use chrono::Utc;
use judgeflow_backend::route_models::devpost_model::{
    DevpostProject,
    DevpostScrapeSseRequest,
    DevpostScrapeResult,
    DevpostUserLookupResult
};
