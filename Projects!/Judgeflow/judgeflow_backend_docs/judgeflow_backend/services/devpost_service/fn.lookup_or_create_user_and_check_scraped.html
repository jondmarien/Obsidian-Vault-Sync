Expand description

Looks up a user by Clerk user ID or email, creates the user if not found, and checks if their Devpost data has already been scraped.

§Arguments

  • http_client - The HTTP client for making requests to Supabase.
  • supabase_url - The Supabase instance URL.
  • supabase_key - The Supabase service role key.
  • clerk_uid - The Clerk user ID to look up.
  • email - The user’s email address to look up.

§Returns

  • Result<DevpostUserLookupResult> - The result containing user lookup information or an error.
