§Discord Webhook Routes

This module provides API endpoints for sending notifications to Discord webhooks. It handles the HTTP interface for Discord integration, including request validation and response formatting.

§Endpoints

  • POST /api/notify/discord: Sends a formatted message to a configured Discord webhook

§Authentication

All endpoints require authentication via the auth_middleware.

§Rate Limiting

Be aware of Discord’s rate limits:

  • 5 requests per 2 seconds per webhook URL
  • 1000 requests per 10 minutes per webhook URL
