ยงRoutes Module

This module contains all the route definitions for the Judgeflow API. Each submodule groups related routes and their handlers. Routes are organized by functionality and resource type.

ยงAvailable Route Modules

  • import_routes: Handles data import functionality (e.g., CSV imports)
  • resource_routes: Generic CRUD operations for various resources
  • waitlist_routes: Waitlist management endpoints
  • webhook_routes: Webhook receivers for external services
  • discord_routes: Discord integration endpoints
  • sync_routes: Data synchronization endpoints
  • devpost_routes: Devpost API integration
  • s3_routes: File upload and management with S3
  • user_routes: User management and authentication
  • status_routes: Health check and system status

ยงUsage

use axum::Router;
use crate::routes::{
    import_routes,
    resource_routes,
    // ... other route modules
};

let app = Router::new()
    .merge(import_routes::router())
    .merge(resource_routes::router())
    // ... merge other routes
