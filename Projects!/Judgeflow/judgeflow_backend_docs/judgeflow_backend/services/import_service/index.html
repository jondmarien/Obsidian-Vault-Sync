§Import Service

This module provides functionality for importing and processing CSV data, specifically for projects and registrants. It handles file uploads, parsing, and data transformation.

§Features

  • CSV file upload and parsing
  • Multipart form data handling
  • Data validation and error handling
  • Integration with database operations

§Usage

use axum::{
    extract::Multipart,
    http::StatusCode,
    response::IntoResponse,
};
use judgeflow_backend::services::import_service::{
    handle_combined_csv_upload,
    CsvProcessingError
};
