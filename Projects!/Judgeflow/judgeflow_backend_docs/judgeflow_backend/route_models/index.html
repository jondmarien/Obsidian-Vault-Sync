ยงRoute Models Module

This module contains all route-specific data models used by the backend API. These models define the request and response structures for various API endpoints.

ยงOrganization

  • devpost_model: Models for Devpost integration and data handling
  • discord_model: Data structures for Discord webhook events and notifications
  • import_model: Models for data import/export functionality
  • resources: Shared resource representations used across multiple routes
  • sync_model: Models for data synchronization operations
  • webhook_model: Webhook payload and event definitions
  • s3_models: S3 storage related data structures
