pub struct DiscordService {

Expand description

Service for sending notifications to Discord channels via webhooks.

This service provides a high-level interface for sending messages to Discord channels. It handles webhook URL configuration, message formatting, and error handling.

§Example

use judgeflow_backend::services::discord_service::DiscordService;
use judgeflow_backend::route_models::discord_model::DiscordMessage;

let service = DiscordService::new();
let message = DiscordMessage {
    content: "Hello from Judgeflow!".to_string(),
    embeds: None,
};
 
service.send_discord_notification(message).await?;

pub async fn send_discord_notification( &self, message: DiscordMessage, ) -> Result<DiscordResponse, String>

Sends a notification to a Discord channel using a webhook.

§Arguments
  • message - The message to send, including content and optional embeds
§Returns
  • Result<DiscordResponse, String> - The result of the operation
§Errors

Returns an error if:

  • The Discord webhook URL is not configured
  • The webhook request fails
  • The message is invalid or too large
§Example
let service = DiscordService::new();
let message = DiscordMessage {
    content: "Hello from Judgeflow!".to_string(),
    embeds: None,
};
 
let response = service.send_discord_notification(message).await?;
println!("Notification sent: {}", response.message);
