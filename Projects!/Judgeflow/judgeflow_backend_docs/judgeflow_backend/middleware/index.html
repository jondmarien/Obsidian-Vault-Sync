§Middleware Module

This module provides middleware components for the Judgeflow backend, primarily focused on request processing, authentication, and authorization. Middleware in this module can be used to intercept and process HTTP requests and responses.

§Available Middleware

  • auth: Handles JWT-based authentication using Clerk

§Usage

Add middleware to your Axum router like this:

use axum::Router;
use crate::middleware::auth::auth_middleware;

let app = Router::new()
    .route("/protected", get(protected_handler))
    .layer(middleware::from_fn(auth_middleware));

§Security Considerations

  • Always ensure proper error handling in middleware
  • Be cautious with error messages to avoid leaking sensitive information
  • Use the development bypass feature only in testing environments
