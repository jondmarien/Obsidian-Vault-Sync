ยงServices Module

This module contains the business logic for the Judgeflow backend. Each submodule corresponds to a specific domain or functionality area of the application.

ยงAvailable Services

  • devpost_service: Handles Devpost profile scraping and data processing
  • discord_service: Manages interactions with the Discord API
  • eligibility_service: Implements eligibility checking logic for hackathon submissions
  • import_service: Handles data import operations
  • resource_service: Manages application resources
  • s3_service: Provides S3 storage integration
  • sync_service: Handles data synchronization between systems
  • user_service: Manages user-related operations
  • waitlist_service: Handles waitlist functionality
  • webhook_service: Processes incoming webhook events
