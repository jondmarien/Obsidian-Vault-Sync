Expand description

Service for managing user data and authentication.

This service provides methods to interact with user data stored in Supabase, including retrieving user information and converting between different user models.

ยงFields

  • supabase_url: Base URL of the Supabase instance
  • supabase_key: API key for authenticating with Supabase

ยงExample

use judgeflow_backend::services::user_service::UserService;

let user_service = UserService::new();
let user = user_service.get_user_by_clerkid("clerk_123").await?;
println!("User email: {}", user.email);
