§Resource Routes

This module provides generic CRUD (Create, Read, Update, Delete) endpoints for various resources. It’s designed to handle common operations on database tables with automatic user ownership checks.

§Endpoints

  • GET /api/{resource}: List all items for the current user
  • POST /api/{resource}: Create a new item
  • PUT /api/{resource}/{id}: Update an existing item
  • DELETE /api/{resource}/{id}: Delete an item

§Authentication

All endpoints require authentication via the auth_middleware.

§Resource Validation

The module enforces resource validation and user ownership for all operations.
