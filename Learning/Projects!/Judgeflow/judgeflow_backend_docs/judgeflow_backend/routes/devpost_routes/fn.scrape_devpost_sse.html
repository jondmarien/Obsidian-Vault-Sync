Scrapes a Devpost profile and streams progress updates via Server-Sent Events (SSE).

§Endpoint

GET /api/devpost/scrape?url=<devpost_url>&user_uuid=<user_uuid>&email=<email>

§Query Parameters

  • url: The Devpost profile URL to scrape
  • user_uuid: The UUID of the user making the request
  • email: (Optional) Email of the user

§Returns

  • 200 OK with an SSE stream of progress events
  • 400 Bad Request for invalid parameters
  • 500 Internal Server Error for scraping failures

§Events

  • Progress messages during scraping
  • Error messages if the operation fails
  • Completion message with results

§Example

curl -H "Authorization: Bearer <token>" \
  "http://localhost:3000/api/devpost/scrape?url=https://devpost.com/username&user_uuid=123e4567-e89b-12d3-a456-426614174000"
