Expand description

Inserts multiple projects into the Supabase database.

This function takes a vector of project records and inserts them into the Supabase database using the REST API. It requires the following environment variables:

  • SUPABASE_URL: The base URL of your Supabase instance
  • SUPABASE_SERVICE_ROLE_KEY: The service role key for authentication

  • projects - A vector of project records to insert

A Result indicating success or containing an error if the operation fails

Returns an error if:

  • Required environment variables are not set
  • The HTTP request fails
  • The API returns an error status
