pub async fn upload_handler(multipart: Multipart) -> Json<Value>
Expand description

Handles file uploads to S3 via multipart form data.

§Endpoint

POST /api/s3/upload

§Request Format

  • Content-Type: multipart/form-data
  • Body: Form with a file part named file

§Returns

  • 200 OK with { "url": "<s3-presigned-url>" } on success
  • 400 Bad Request if no file is provided or upload fails

§Example

curl -X POST \
  -H "Authorization: Bearer <token>" \
  -F "file=@/path/to/image.jpg" \
  http://localhost:3000/api/s3/upload

§Example Response

{
  "url": "https://bucket.s3.region.amazonaws.com/path/to/uploaded_file.jpg"
}
