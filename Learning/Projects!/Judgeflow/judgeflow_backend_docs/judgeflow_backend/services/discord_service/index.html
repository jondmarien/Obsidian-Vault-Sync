§Discord Service

This module provides a service for sending notifications to Discord channels via webhooks. It handles the creation and sending of Discord messages with support for rich embeds, error handling, and environment-based configuration.

§Features

  • Send text messages to Discord channels
  • Support for rich embeds in messages
  • Environment-based webhook URL configuration
  • Error handling and status code mapping

§Configuration

The service requires the following environment variable:

  • DISCORD_WEBHOOK_URL: The Discord webhook URL to send messages to

§Example

use judgeflow_backend::services::discord_service::DiscordService;
use judgeflow_backend::route_models::discord_model::DiscordMessage;

#[tokio::main]
async fn main() {
    let service = DiscordService::new();
    let message = DiscordMessage {
        content: "Hello from Judgeflow!".to_string(),
        embeds: None,
    };
     
    match service.send_discord_notification(message).await {
        Ok(response) => println!("Notification sent: {}", response.message),
        Err(e) => eprintln!("Failed to send notification: {}", e),
    }
}
