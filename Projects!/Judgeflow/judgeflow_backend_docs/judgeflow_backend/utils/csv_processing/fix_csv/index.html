Â§CSV File Repair and Validation

This module provides functionality to clean, validate, and repair malformed CSV files. It can handle various CSV formatting issues including:

  • Inconsistent quoting
  • Missing or extra fields
  • Malformed quoted fields
  • Inconsistent line endings
  • Comment lines

Â§Features

  • Automatic detection and repair of common CSV formatting issues
  • Support for quoted fields containing commas and newlines
  • Flexible handling of different CSV dialects
  • Preservation of data integrity during repair

Â§Example

use judgeflow_backend::utils::csv_processing::fix_csv;
use std::path::Path;

// Fix a malformed CSV file
let input_path = "data/malformed.csv";
let output_path = "data/fixed.csv";
 
// Fix the CSV and save to a new file
fix_csv::fix_csv(input_path, output_path)?;
 
println!("CSV file has been repaired and saved to {}", output_path);
