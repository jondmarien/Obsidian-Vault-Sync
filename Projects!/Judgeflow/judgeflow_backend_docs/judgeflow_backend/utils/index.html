§Utilities Module

This module contains various utility functions and types used throughout the Judgeflow backend. It’s organized into submodules, each focusing on a specific area of functionality.

§Available Submodules

  • csv_processing: Tools for processing CSV files, including reading, writing, and transforming CSV data.
  • eligibility: Functionality for checking participant eligibility based on various criteria.
    • eligibility_checks: Core logic for individual eligibility checks.
    • eligibility_csv: Batch processing of eligibility checks from CSV files.

§Usage

Import the specific utilities you need in your module:

// For CSV processing
use judgeflow_backend::utils::csv_processing::{read_csv, write_csv};

// For eligibility checking
use judgeflow_backend::utils::eligibility::{
    EligibilityCheckInput,
    eligibility_checks::perform_all_checks,
    eligibility_csv::check_eligibility_from_csv_and_save
};

§Features

  • CSV Processing: Read and write CSV files with automatic type conversion
  • Eligibility Checking: Validate participant eligibility based on configurable rules
  • Batch Processing: Process multiple participants efficiently from CSV files

§Error Handling

Most functions in this module return Result types with appropriate error variants. Always handle potential errors when using these utilities.
