Expand description

Sends a message to a configured Discord webhook.

§Endpoint

POST /api/webhooks/discord

§Authentication

Requires DISCORD_WEBHOOK_URL environment variable to be set

§Request Body

  • DiscordMessage: The message content to send to Discord

§Returns

  • 200 OK with success status on successful send
  • 400 Bad Request for invalid requests
  • 500 Internal Server Error if webhook URL is not configured

§Example Request

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"content":"Hello from Judgeflow!"}' \
  http://localhost:3000/api/webhooks/discord
