§Webhook Models

This module defines data structures for handling both incoming and outgoing webhook events. It includes models for authentication provider webhooks (like Clerk) and external service webhooks (like Discord).

§Key Components

  • ClerkWebhookEvent: Enum representing different types of Clerk webhook events
  • UserCreatedData/UserUpdatedData: User-related webhook payloads
  • DiscordMessage: Structure for sending messages to Discord webhooks
  • WebhookResponse: Standardized response format for webhook handlers

§Usage

use judgeflow_backend::route_models::webhook_model::{
    ClerkWebhookEvent,
    UserCreatedData,
    DiscordMessage,
    WebhookResponse
};
