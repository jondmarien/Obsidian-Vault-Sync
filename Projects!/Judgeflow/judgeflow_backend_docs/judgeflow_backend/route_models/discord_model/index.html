§Discord Integration Models

This module defines data structures for interacting with Discord’s webhook API and handling Discord-specific events within the application.

§Key Components

  • DiscordResponse: Standardized response format for Discord operations
  • Re-exports DiscordMessage from webhook_model for message formatting

§Usage

use judgeflow_backend::route_models::discord_model::DiscordResponse;

// Create a success response
let response = DiscordResponse {
    success: true,
    message: "Notification sent successfully".to_string(),
};
